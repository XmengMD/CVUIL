# CVUIL — ConVenient UI Library

[![C++11](https://img.shields.io/badge/C%2B%2B-11%2B-blue.svg)](https://isocpp.org/)
[![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)](https://www.microsoft.com/windows/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

CVUIL 是一个自定义的 WINAPI 编程库，旨在简化 WINAPI 的调用，使基础 Windows 程序开发更加简单易用。通过 CVUIL，开发者可以快速创建和管理各种控件，而无需深入复杂的 WINAPI 细节。

## ✨ 特性

- **简洁的 API** — 以组件化方式创建和管理 Windows 控件，大幅降低基础 Windows 程序 的开发成本
- **完整的基础控件集** — 按钮、复选框、单选框、输入框、组合框、列表、表格、树形控件、滑块、进度条、菜单栏等
- **增强功能** — 输入框增强（密码、数字限制）、文件对话框、工具提示、异步任务支持
- **特性兼容** — 基于 C++ 封装，与原生 C/C++ 代码及 WINAPI 函数无缝共存

在 CVUIL/User_manual/CVUIL_Instruction_manual.md 阅读 CVUIL 的使用说明和示例代码。
